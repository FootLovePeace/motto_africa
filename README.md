# 概要
## ユーザー登録すると、アフリカ大陸の全ての国について様々なジャンルで自由に記事を投稿したり、記事に対してコメントすることができます。<br>また、ログイン状態に関わらず、ユーザーは自由に記事を閲覧したり、「国名」と「ジャンル」を組み合わせて、記事を検索することもできます。

# 制作背景
### 昔アフリカでボランティア活動をしたことがありましたが、それまでは「アフリカ＝未知数」というイメージでした。しかし、実際に現地で生活してみると良い意味でイメージが覆されることが沢山あり、現地での活動を通してアフリカが身近に感じられるようになりました。<br>この経験から、もっとアフリカについて知ることができれば、アフリカについての関心や興味も湧いて身近なものになると思いました。<br>そこで、アフリカが好きな人や興味はあるけどイメージがわかない人にとって、「もっとアフリカを身近に感じることができる」をコンセプトにした、アフリカの各国の様々な情報がまとまっている投稿サイトを制作しようと思いました。
<br>

# 要件定義
1. ### ユーザー管理機能
   * 新規登録画面からフォームを入力してユーザー新規登録する。
   * ログインする際はログインボタンからフォームを記入してログインする。
   * ログアウトボタンを押すとログアウトする。
   * フォームの入力が正しくないとエラーメッセージを表示する。
2. ### 記事投稿機能
   * ログイン済みのユーザーは「投稿する」ボタンからフォームを記入して記事を新規投稿できる。
   * フォームの入力が正しくないとエラーメッセージを表示する。
3. ### 投稿一覧表示機能
   * 投稿済みの記事はトップページに一覧で表示される。
   * 表示される順番は降順で新しいものが上に表示される。
4. ### 投稿詳細表示機能
   * 記事が投稿されていることが前提。
   * ログイン状態に関わらず、ユーザーは「記事を読む」をクリックすると、その記事の詳細ページに遷移して本文を読むことがきる。（一覧表示の時点では「本文」と「コメント」以外の内容が表示されるので、詳細ページで初めて記事の本文を読むことができる）
5. ### 投稿情報編集機能
   * 記事が投稿されていることが前提。
   * 記事の投稿者のみ、記事の詳細ページから「編集」ボタンを押すと、記事を編集することができる。
   * 編集したら「変更する」ボタンを押すと編集内容が反映され、詳細ページに戻る。
6. ### 投稿削除機能
   * 記事が投稿されていることが前提。
   * 記事の投稿者のみ、記事の詳細ページから「削除」ボタンを押すと、記事を削除することができる。
   * 削除後はトップページに戻る。
7. ### 投稿検索機能
   * 記事が投稿されていることが前提。
   * ログイン状態に関わらずユーザーは「国名」と「ジャンル」を指定して「検索」ボタンを押すと記事を検索することができる。（指定するのはどちらか一方でも可能）
   * 該当する記事を検索結果ページで表示する。
8. ### コメント投稿機能
   * 記事が投稿されていることが前提。
   * ログイン済みのユーザーのみ、記事の詳細ページからフォームに記入して「コメントする」ボタンを押すとコメントすることができる。
9. ### ユーザー詳細機能
   * ログインしているユーザーは、トップページのフッターに表示される「アカウント名（自身のニックネーム）」、記事の詳細ページの「◯◯さんの投稿した記事の一覧へ」、コメントがあればコメントの「投稿者」ボタンを押すと、ユーザー詳細ページをみることができる。
   * ログインしていないユーザーでも、記事の詳細ページの「◯◯さんの投稿した記事の一覧へ」とコメントがあればコメントの「投稿者」ボタンを押すと、ユーザー詳細ページをみることができる。
   * ユーザー詳細ページへ遷移すると、そのユーザーが投稿した記事の一覧が表示される。
10. ### ページネーション機能
   * 記事が複数投稿されていることが前提。
   * ユーザーは投稿された記事の一覧をページを移動しながらみることができるので、分割することで快適に利用できる。
